# AI Engineer Curriculum — 4 Months to $130K–$250K+

An interactive React app that maps out a complete 4-month curriculum to become a job-ready AI Engineer. Built with real 2026 job market data and salary research.

## Quick Start

```bash
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

## What's Inside

| Tab | Description |
|-----|-------------|
| **Overview** | Career paths, salaries, daily schedule, 4-month roadmap |
| **Curriculum** | Day-by-day breakdown with expandable phases, topics, subtopics |
| **DSA Track** | 120 LeetCode problems organized by pattern (runs parallel) |
| **How Much?** | Exact targets per category — when you're done vs overkill |
| **Portfolio** | 5 project specs with must-haves and deployment targets |
| **Interview** | 60+ questions across ML, DL, LLMs, RAG, Agents, MLOps |
| **Resources** | Books, YouTube, courses, communities, newsletters, tools |

## 4-Month Roadmap

- **Month 1** — Python speed-run, Math for ML, Classical ML with Scikit-Learn, Kaggle
- **Month 2** — Deep Learning (PyTorch), LLMs, Prompt Engineering, RAG (basic to production)
- **Month 3** — AI Agents (LangGraph, CrewAI, MCP), Fine-Tuning (LoRA/QLoRA), Multimodal AI, MLOps
- **Month 4** — AI System Design, Flutter + AI, Portfolio Sprint, Interview Prep, Job Hunt
- **Parallel** — 1h/day DSA (120 problems, pattern-based, NeetCode roadmap)
